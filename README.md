## 🧠 What’s This?

A clean, blazing-fast frontend built with **React + Vite** to power your Gen‑AI ticketing workflow.

This UI lets users:
- Submit tickets 🔧
- View ticket lists 📄
- Handle assignments & statuses 🎯
All connected to an AI brain running in the backend (separate repo).

---
